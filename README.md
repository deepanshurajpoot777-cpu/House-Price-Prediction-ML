# House-Price-Prediction-ML
A Machine Learning project that predicts house prices using Linear Regression with data preprocessing, feature scaling, model evaluation, visualization, and user input prediction.

#  House Price Prediction using Machine Learning

#  Project Overview
This project predicts house prices using the Linear Regression algorithm. It demonstrates the complete machine learning workflow, including data preprocessing, feature engineering, model training, evaluation, visualization, and prediction based on user input.

#  Features
- Data Loading and Exploration
- Data Cleaning
- Label Encoding
- One-Hot Encoding
- Train-Test Split
- Feature Scaling
- Linear Regression Model
- Model Evaluation
- Data Visualization
- User Input House Price Prediction

#  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

#  Dataset
Housing_price_Dataset.csv

#  Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

#  Visualization
Actual House Price vs Predicted House Price Scatter Plot

#  How to Run
1. Install required libraries.
2. Place Housing.csv in the project folder.
3. Run the Python script.
4. Enter house details when prompted.
5. View the predicted house price.
